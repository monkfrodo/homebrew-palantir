# Homebrew Palantir

## O que é
Homebrew tap para instalação do Palantir (live wallpapers macOS).

## Stack
- Ruby (Homebrew formula)

## Estrutura
```
Formula/palantir.rb  ← fórmula Homebrew
```

## Repositório relacionado
- App principal: github.com/monkfrodo/palantir

## Notas
- A fórmula aponta para a tag `v1.0` do repo principal
- Post-install executa `install.sh` automaticamente
- Atualizar a tag na fórmula quando houver nova versão do Palantir

## Git
- Branch: main
